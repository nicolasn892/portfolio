# Análise de Turismo em Portugal — Demo

Este projeto demonstra uma pequena análise interativa de dados de turismo por cidade.
Funciona diretamente no GitHub Pages (HTML + Plotly.js) sem backend.

Como usar:
1. Abra `index.html` no navegador (ou pelo GitHub Pages).
2. O arquivo `data/turismo.csv` contém os dados de exemplo.

Tecnologias:
- HTML / JavaScript
- Plotly.js
- CSV simples (dados de exemplo)
